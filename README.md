# AZGen-web

## Local development

In App.vue, set the POST request URL to `localhost:8082` and make sure [azgen-api](https://github.com/arkorobotics/azgen-api) is running.
```
.post('http://localhost:8082', data)
//.post('https://api.activation.zone', data)
```
